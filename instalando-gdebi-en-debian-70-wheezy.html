Instalando GDebi en Debian 7.0 Wheezy

Wed 19 November 2014
By Jhon Ramirez

Que onda!

Hoy vamos a ver como instalar GDebi, pero ¿Qué es eso?.

GDebi es una aplicación muy sencilla para instalar aplicaciones en las distribuciones (distros) de GNU/Linux basadas en Debian a través de un archivo ".deb". Ok, pero ¿Qué es un archivo .deb?. Bueno pues un archivo .deb es la extensión para el formato de paquetes de software de Debian, es decir, es un archivo con todo lo necesario para instalar una aplicación específica en un computador que corra Debian o uno de sus derivados.

GDebi permite instalar estas aplicaciones de manera local (esto significa que la instalación se hace a partir del archivo), resolviendo e instalando sus dependencias (librerias necesarias para que todo funcione bien) automáticamente, a diferencia del comando "apt" que hace lo mismo pero solo para paquetes remotos (generalmente almacenados en repositorios).

Pero, si cada distro de GNU/Linux viene con un administrador de paquetes, el cual permite al usuario buscar e instalar (en Debian se llama Synaptic) el paquete que desee, ¿Para qué quiero instalar una aplicación que también instala paquetes?. Pués resulta que instalar un paquete con Synaptic, es igual que instalarlo con el comando "apt", puesto que este administrador busca el paquete y sus dependencias en los repositorios de la distro. La gran diferencia es que GDebi instala el paquete que se le indique, el cual debe ser un archivo con extensión ".deb".

Ahora bien, ¿Cómo instalar GDebi?

  1. Abrir una terminal e iniciar sesión como root. (Esto se hace con el comando "su" seguido de introducir la contraseña de administrador del sistema).
  2. Ingresar el siguiente comando: "apt-get install gdebi".
  3. En un momento el sistema va a preguntar si queremos descargar una cantidad específica de MB (los archivos de instalación) y luego de ellos crear unos archivos que ocuparán otra cantidad específica de MB en el disco duro. Por ello para continuar con la instalación debemos presionar la tecla "Y" de la palabra "Yes".
  4. Esperar que termine y listo!

A continuación encontrarán el video de la instalación de GDebi.

Con eso ya tenemos instalado esta aplicación para instalar aplicaciones (jajajaj), pero, ¿Cómo instalo una aplicación con GDebi?. Pués lo único que se debe hacer es hacer click derecho sobre el paquete ".deb" y seleccionar la opción "Open with GDebi Package Installer" o "Abrir con Instalador de Paquetes GDebi" si el lenguaje de la distro es español.
